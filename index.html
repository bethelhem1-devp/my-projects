Course Descriptions

Networking and Communication

This course introduces the fundamentals of computer networks, communication protocols, and data transmission. You’ll learn about network models, IP addressing, routing, and how devices communicate securely across the internet.

Data Structures and Algorithms

Explore the building blocks of efficient programming. This course covers arrays, linked lists, stacks, queues, trees, graphs, and algorithms for searching and sorting. It emphasizes problem-solving and optimization techniques used in real-world applications.

Computer Architecture and Organization

Dive into the inner workings of computers. Topics include CPU design, memory hierarchy, instruction sets, and input/output systems. You’ll understand how hardware and software interact to execute programs efficiently.
